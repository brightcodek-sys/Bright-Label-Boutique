# BRIGHT LABEL BOUTIQUE

A high-end, minimalist e-commerce Progressive Web App (PWA) built with Tailwind CSS and Firebase.

## 🚀 Quick Start
1. **Firebase Setup**: Create a Firebase project and enable Firestore, Auth, and Storage.
2. **Admin Email**: Update `const MY_ADMIN_EMAIL` in `index.html` to your own email.
3. **Files**: Keep `index.html`, `manifest.json`, and `sw.js` in the same folder.
4. **Hosting**: Deploy to Firebase Hosting, GitHub Pages, or any HTTPS server.

## 📱 Mobile App Features
- **Installable**: Adds to home screen with a custom icon.
- **Responsive**: Works perfectly on mobile, tablet, and desktop.
- **Real-time**: Stock updates instantly when someone makes a purchase.

## 🛠 Tech Stack
- **Styling**: Tailwind CSS
- **Database**: Google Firebase Firestore
- **Storage**: Google Firebase Storage
- **Auth**: Firebase Authentication

## 📂 File Structure
- `index.html`: The main UI and logic.
- `manifest.json`: Mobile app identity configuration.
- `sw.js`: Service worker for PWA installation.
